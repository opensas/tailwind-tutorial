# First steps with figma, tailwind & SvelteKit

Sample application to learn tailwind and SvelteKit

## Create the project

```shell
pnpm init svelte@next tailwind-tutorial

cd tailwind-tutorial

pnpx svelte-add@latest tailwindcss

pnpm install

pnpm run dev -- --open
```
